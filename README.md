-Abre tu terminal de VS Code) e instala Streamlit con el siguiente comando:

-pip install streamlit

-Ejecuta la aplicación con Streamlit:

-streamlit run app.py

-¡Listo! Automáticamente se abrirá una pestaña en tu navegador web (normalmente en la dirección http://localhost:8501) con tu sistema de "Captcha Vial y Urbano".

-Para la opción "Olvidé mi contraseña", se debe ingresar el nombre de usuario. Luego, el sistema generará un código de recuperación, el cual se mostrará en la terminal de Visual Studio Code.

-Una vez obtenido el código, se debe volver a seleccionar la opción "Olvidé mi contraseña" e ingresar el código generado.

-Después de validar el código, el sistema mostrará la contraseña asociada a la cuenta.
